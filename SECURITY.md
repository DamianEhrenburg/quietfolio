# Security Policy

## Supported Version

Quietfolio is early desktop software. Security fixes target the latest tagged version and the `main` branch.

## Reporting

Please do not open a public issue for sensitive security reports.

Send a private report to Damian Ehrenburg through the contact links on the GitHub profile, or open a minimal issue asking for a private contact channel if no direct contact is available.

## Local Data

Quietfolio stores library data locally in Electron `userData`. Optional API keys are stored through Electron `safeStorage` when encryption is available.
