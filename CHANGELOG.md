# Changelog

## [0.5.0] - 2026-06-17

### Added

- Electron app with React UI and SQLite storage
- Reading statuses, star ratings, reviews, cover cache
- Online search: FantLab, Google Books, Wikidata, Open Library
- Russian and English UI, language picker on first launch
- Favorites filter and detail-panel toggle
- Show-more in search, sorting, locale-aware results, quick-add

### Changed

- Home layout, library grid, book details, search UX
- Author lookup for personal names, faster catalog queries
- Cleaner descriptions in search results

### Not yet

- Import and export (menu placeholders only)
